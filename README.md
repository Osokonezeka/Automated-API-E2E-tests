# *Automated API / E2E tests*

***

> [!NOTE]
> This project was created to showcase skills in Python automation (at the request of one of the companies).

***

## Purpose

The purpose of this project is to:
* Check the magnifying glass on the home page.
* Search for the ***phrase*** and check if it appears in the search results and on the movie page in H1 (if it exists).
* Check if the video player is visible on the page and if the movie can be played.
* Async verification: Check if playback is interrupted within 1 to 60 seconds and a popup appears on the screen, as well as check which page **(URL)** it will take the user to.
